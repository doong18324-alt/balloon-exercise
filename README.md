# 🎈 AI 맞춤형 운동 트레이너

ACSM 기반 개인 맞춤 운동 프로그램 + 풍선 터트리기 운동 + PDF 리포트

## 주요 기능

- **개인 프로필 입력**: 연령, 키, 체중, 성별, 기저질환 체크리스트
- **ACSM 위험도 평가**: 저/중/고위험 분류, 목표 심박수 산출
- **밴드 색상 추천**: 연령·체중·질환 기반 5단계 저항 밴드 추천
- **맞춤 운동 프로그램**: 5/10/15/20/30분, 밴드/맨몸 선택
- **질환별 운동 제한**: 무릎통증→의자스쿼트, 어깨통증→오버헤드 제외 등
- **풍선 터트리기 운동**: MediaPipe 포즈 감지 + 풍선 팝 효과
- **식이 권장사항**: BMI·질환 기반 칼로리, 단백질, 영양소 가이드
- **PDF 리포트**: 전체 평가·운동·식이 결과를 PDF로 다운로드

## 운동 콘텐츠

### 상체 밴드운동 (6종)
Biceps Curl, Shoulder Press, Lateral Raise, Front Raise, Pull-Apart, Upright Row

### 상체 맨몸운동 (4종)
Arm Circles, Shoulder Taps, Overhead Reach, Wall Push-Up

### 하체 운동 (4종)
Squat, Chair Squat, Calf Raises, Side Steps

### 워밍업/쿨다운
Marching in Place, Deep Breathing
