# 🎈 풍선 터트리기 운동 트레이너

AI 카메라 기반 실시간 포즈 감지로 풍선을 터트리며 운동하는 웹 앱

## 주요 기능

- **실시간 포즈 감지**: Google MediaPipe Pose Landmarker (33개 관절)
- **상체 밴드운동 5종**: Biceps Curl, Shoulder Press, Lateral Raise, Front Raise, Overhead Pull-Apart
- **하체 스쿼트 운동 1종**: Bodyweight Squat
- **풍선 터트리기 효과**: 운동 부위에 풍선 표시 → 동작 완료 시 터짐 + 파티클 효과 + 사운드
- **각 운동 10회 반복**, 총 60회 (전체 모드)

## 사용 방법

1. 웹사이트 접속
2. 운동 모드 선택 (상체 / 하체 / 전체)
3. 카메라 권한 허용
4. 화면의 풍선을 향해 동작 수행!

## 지원 환경

- iPhone Safari ✅
- Android Chrome ✅
- PC Chrome / Edge / Firefox ✅
- 카메라가 있는 모든 기기

## 기술 스택

- HTML / CSS / JavaScript (순수 웹, 프레임워크 없음)
- Google MediaPipe Tasks Vision (Pose Landmarker Lite)
- Web Audio API (팝 사운드)
- Canvas 2D (풍선, 파티클, 스켈레톤 렌더링)
